Core Goal - This project gives the user to see underlying trends in the way they behave.

Must-Have features -
Input of user data
Ability to store the data
Being able to create 1 or 2 graphs about the data

Out of scope -
AI Insights
Improved UI capabilities
Addition of several graphs

Success Criteria - This project will be deemed a success when the user can add data and it will be stored. The data will be stored somewhere where the user will be able to interact with it and display it on their screen. The data will be able to be visualised in at least one chart that works.
