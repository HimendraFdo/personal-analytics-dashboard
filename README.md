# Personal Analytics Dashboard

A full-stack dashboard for tracking habits and visualising trends from personal data.

## Features
- Add and manage entries
- Visualise trends and summaries
- Local data persistence
- Responsive UI

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite

## Why I Built It
I built this project to strengthen my frontend development skills and practise designing reusable components, handling state, and turning raw data into useful visual insights.

## Running the Project
1. Clone the repo
2. Install dependencies with `npm install`
3. Start the dev server with `npm run dev`

## What I Learned
- Component architecture
- State and data transformation
- Debugging frontend issues
- Building cleaner UI layouts
