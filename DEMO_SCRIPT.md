# 60-second demo script

1. **Open the live dashboard** — Say: “This is my personal analytics app: I log structured entries and see trends on one screen.”
2. **Dashboard (`/dashboard`)** — Point at summary cards and recent entries. Say: “Totals and category breakdowns update from PostgreSQL via the Next.js API.”
3. **Entries (`/entries`)** — Add a new row (title, value, category, date). Say: “CRUD goes through REST routes with Zod validation before Prisma writes.”
4. **Analytics (`/analytics`)** — Open the chart view. Say: “Recharts visualises the same dataset—no duplicate client-only storage.”
5. **Close** — Say: “Stack is Next.js, Prisma, PostgreSQL on Vercel + Neon; CI runs lint, tests, and build on every PR.”
